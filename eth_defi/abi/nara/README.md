# Nara ABI sources

`NaraUSDPlus.json` contains the NaraUSD+-specific cooldown interface recovered
from the public [Nara application](https://app.nara.io/swap) and verified against
the deployed [Ethereum contract](https://etherscan.io/address/0x1aa23CDFC941f6b54251C72012A9Bfa4bF5394D6).

The shared ERC-4626 interface is loaded separately by the vault base class.
