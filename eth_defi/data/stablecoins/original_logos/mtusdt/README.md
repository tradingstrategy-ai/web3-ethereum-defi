# mtusdt (Mutuum Finance USDT) — logo sources

## Project

Mutuum Finance is a decentralised, non-custodial lending and borrowing protocol on Ethereum.
- Homepage: https://www.mutuum.com/
- Docs: https://docs.mutuum.com/
- Twitter/X: https://x.com/MutuumFinance
- GitHub: https://github.com/mutuumfinance

Note: mtusdt and mtusdc are both issued by Mutuum Finance and share the same brand.
The original domain `mutuum.finance` was not resolving at the time of collection (2026-03-17);
logos were sourced from `mutuum.com`.
