# NaraUSD logo sources

## Source

- Official Nara brand kit: https://www.nara.io/brand
- NaraUSD SVG token mark: https://www.nara.io/brand/logos/NaraUSD.svg
- NaraUSD PNG token mark: https://www.nara.io/brand/logos/NaraUSD.png

The official brand kit identifies these as the NaraUSD stablecoin token mark.

## Files

- `narausd.generic.svg` - Official vector NaraUSD token mark.
- `narausd.generic.png` - Official 293x293 NaraUSD token mark.

The PNG is a square icon-only asset and is used to create the formatted
stablecoin logo.
