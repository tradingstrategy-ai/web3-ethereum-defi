# USDXL (Last USD / HypurrFi) logo sources

## Files

- `usdxl.generic.svg` — HypurrFi paw brand mark SVG from the HypurrFi app

## Sources

- **Direct file URL**: https://app.hypurr.fi/hypurr-paw.svg
- **Project homepage**: https://hypurr.fi/
- **CoinGecko**: https://www.coingecko.com/en/coins/last-usd

## Notes

USDXL does not have a dedicated token-specific logo. The HypurrFi paw brand mark is the
official logo for the HypurrFi protocol, which issues USDXL on HyperEVM.
The SVG is 32×26 pixels (non-square), white and pink (#FFB1B5) on transparent background.
The post-processing script will pad it to square before scaling to 256×256.
