# flexUSD logo sources

flexUSD was a yield-bearing stablecoin issued by CoinFLEX, which collapsed in 2022.
The flexusd.com and flexusd.pages.dev domains have since been hijacked by a gambling site.

## Sources

### flexusd.generic.jpg

- **Source**: Twitter/X avatar of the official `@flexUSD` account
- **URL**: `https://unavatar.io/twitter/flexUSD` (proxy to X avatar)
- **Original account**: https://x.com/flexUSD
- **Resolution**: 400×400 px, JPEG
- **Notes**: Best available source for the flexUSD-specific token logo given the domain hijacking.

### coinflex.wordmark.light.svg

- **Source**: CoinFLEX GitHub repository (CoinFLEX API documentation)
- **URL**: https://raw.githubusercontent.com/coinflex-exchange/v2/master/source/images/logo.svg
- **Repository**: https://github.com/coinflex-exchange/v2
- **Description**: Full CoinFLEX wordmark (spiral icon + "COINFLEX" text), white/light on transparent background
- **Dimensions**: 4024×953 viewBox

### coinflex.icon.light.svg

- **Source**: Extracted from `coinflex.wordmark.light.svg`
- **Description**: Just the spiral/ring brandmark portion of the CoinFLEX logo, white on transparent background
- **Dimensions**: 953×953 viewBox

## Notes

- CoinFLEX collapsed in June 2022 after halting withdrawals; all official web properties were later abandoned or taken over
- The CoinFLEX GitHub organisation remains at https://github.com/coinflex-exchange
- CoinGecko page: https://www.coingecko.com/en/coins/flex-usd (may still have logo)
- DeFiLlama stablecoin entry: https://defillama.com/stablecoins (id: 21)
