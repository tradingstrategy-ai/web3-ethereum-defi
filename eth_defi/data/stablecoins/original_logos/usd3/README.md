# 3Jane (USD3 / sUSD3) logo sources

Logos for the 3Jane protocol, issuer of the USD3 and sUSD3 stablecoins.

## Source

- Website: <https://www.3jane.xyz/>
- Twitter/X: <https://x.com/3janexyz>
- Docs: <https://docs.3jane.xyz/>

All assets were taken directly from the official website's static assets.

## Files

| File | Source URL | Description |
|------|-----------|-------------|
| `logo.png` | `https://www.3jane.xyz/logo.png` | 4000×4000 RGBA brand mark — white pixel-art face on 3Jane blue (`#0029FF`) circle. Square; used as the post-processing input. |
| `logo-mark.svg` | `https://www.3jane.xyz/assets/capital/logo-mark.svg` | Vector brand mark, white fill (light variant), narrow aspect (~10.4×17). |
| `logo-ring.svg` | `https://www.3jane.xyz/assets/capital/logo-ring.svg` | Vector ring element, `#0029FF` (~31×31, near-square). |
| `logo-wordmark.svg` | `https://www.3jane.xyz/assets/capital/logo-wordmark.svg` | Vector "3Jane" wordmark, `#0029FF`. |

## Brand colour

- Primary blue: `#0029FF`

## Notes

`logo.png` is the colour brand mark and is the preferred square source for the
256×256 `formatted_logos/usd3/light.png`. The same mark is reused for sUSD3 (a
staked derivative of USD3 with no distinct mark).
