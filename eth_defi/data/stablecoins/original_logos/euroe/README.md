## Notes

The EUROe website (euroe.com) does not publish a standalone downloadable logo or brand kit.
The apple touch icon (180×180 grayscale PNG) and the safari pinned tab SVG are the only
vector/high-res assets served directly by the site. The CMC 200×200 RGBA PNG provides
the best quality colour version of the logo mark.

The Sushiswap token list also references a 128×128 JPEG:
https://raw.githubusercontent.com/sushiswap/list/master/logos/token-logos/token/euroe.jpg
