# usd8 logo sources

USD8 is a stablecoin project at usd8.io. The site is built on Wix.

## Source

- **Protocol**: USD8
- **Protocol website**: https://usd8.io/
- **Logo source**: Wix static media CDN (usd8.io site assets)
- **Source URL**: https://static.wixstatic.com/media/d5191c_b22d8116ca8f4dc5901e0464e843e8e8~mv2.png
- **Original filename hint**: `usd8logo1.png` (from Wix URL parameters)
- **Format**: PNG, 500×500 px, RGBA with transparency

## Files

- `usd8.generic.png` — USD8 brand mark (gold and silver shield with interlocked "S" letter mark)

## Notes

- A wordmark variant also exists with "USD8" text below the shield:
  https://static.wixstatic.com/media/d5191c_c0ebe4313932478da39770c7b8a03eeb~mv2.png
