# tor logo sources

TOR is a defunct stablecoin issued by Hector Finance (Hector Network) on the Fantom blockchain.
The project collapsed in July 2023 following losses from the Multichain bridge exploit.

## Source

- **Protocol**: Hector Finance / Hector Network
- **Protocol website**: https://hector.network/ (domain expired, now a GoDaddy parked page)
- **Logo source**: Trust Wallet community assets repository (Fantom chain assets)
- **Source URL**: https://raw.githubusercontent.com/trustwallet/assets/master/blockchains/fantom/assets/0x74E23dF9110Aa9eA0b6ff2fAEE01e740CA1c642e/logo.png
- **Token address (Fantom)**: 0x74E23dF9110Aa9eA0b6ff2fAEE01e740CA1c642e
- **Format**: PNG, 256×256 px, RGBA with transparency

## Files

- `tor.generic.png` — TOR brand mark (brown/tan circle with white geometric arrow shapes)

- The official hector.network domain has expired and redirects to GoDaddy parking.
