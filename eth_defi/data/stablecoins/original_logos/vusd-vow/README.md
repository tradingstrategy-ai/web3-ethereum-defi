# VUSD Vow (Vow USD) logo sources

Vow USD (VUSD) is a voucher currency / stablecoin issued by the Vow ecosystem.
VOW aims to become a reserve currency for the global discount voucher trade.

Website: <https://vow.foundation/>
CoinGecko coin ID: `vow-usd`

## Files

- `vusd-vow.generic.png` — 250×250 grayscale PNG from CoinGecko (`vow-usd`).
  Shows "v$" text mark in dark/black on white background. Used for post-processing.
  Note: this is a dark logo variant (dark text on light background).
- `vusd-vow.generic.svg` — Full word mark SVG (150×43) from vow.foundation website.
  Contains the colourful VOW heart-V brand mark plus dark "vow" text word mark.
  Source URL: `https://cdn.prod.website-files.com/6489b3749708241535260c80/649dd989d73cf472f6c058cd_vow_trademark.svg`

## Notes

The Vow USD token uses a minimalist "v$" text mark as its specific token logo.
The broader Vow Foundation uses a colourful rainbow heart-V mark for the VOW currency brand.
No high-resolution coloured VUSD-specific token logo was found at the time of collection.
