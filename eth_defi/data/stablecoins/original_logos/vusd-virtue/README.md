# VUSD Virtue (Virtue USD) logo sources

Virtue USD (VUSD) is a stablecoin issued by Virtue Money on the IOTA network.

Website: <https://virtue.money/>
CoinGecko coin ID: `virtue-usd`

## Files

- `vusd-virtue.generic.svg` — Full word mark SVG from virtue.money homepage (126×48).
  Contains the circular brand mark (left side, 48×48) plus the "Virtue" text word mark.
  White paths on transparent background (designed for dark backgrounds).
- `vusd-virtue.icon.svg` — Extracted brand mark only (48×48 circle with V+drop icon).
  White paths on transparent background. Used for post-processing.
- `vusd-virtue.coingecko.png` — 250×250 RGBA PNG from CoinGecko (`virtue-usd`).
  Shows a blue circle with a "$" mark — differs from the official virtue.money branding.
  Kept as reference but not used for post-processing.
