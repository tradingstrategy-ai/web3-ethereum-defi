# VUSD Vesper (V-Dollar) logo sources

VUSD (V-Dollar) is a stablecoin issued by Vesper Finance on Ethereum mainnet
(contract `0x677ddbd918637E5F2c79e164D402454dE7dA8619`). The project is largely
defunct as of 2026 with ~$113K TVL.

Note: CoinGecko lists this as `vesper-vdollar` and shows the general Vesper VSP token
logo (Vesper_Token_600.png) rather than a VUSD-specific logo. The VirtualDollar logo
from the vesperfi/doc GitHub repository is the correct VUSD branding.

## Files

- `vusd-vesper.generic.png` — 801×800 RGBA PNG from the official Vesper Finance
  GitHub repository (`vesperfi/doc`). Shows a blue/purple circle with a stylised
  V+dollar-lines icon. This is the VirtualDollar (V-Dollar) specific logo.
