# plusd (Trevee Plasma USD) — logo sources

## Project

Trevee Plasma USD (plUSD) is a stablecoin on Plasma blockchain issued by Trevee,
a DeFi protocol that includes Trevee Earn (formerly Rings) and Trevee Quest (formerly Warden Quest).
- App: https://plasma.earn.trevee.xyz/earn/
- CoinGecko: https://www.coingecko.com/en/coins/trevee-plasma-usd
- DeFiLlama: https://defillama.com/protocol/trevee-earn
- Twitter/X: https://x.com/Trevee_xyz
- Contract on Plasma: 0xf91c31299e998c5127bc5f11e4a657fc0cf358cd

Note: `trevee.com` was a parked/for-sale domain at the time of collection (2026-03-17).
The actual app is hosted on `plasma.earn.trevee.xyz`.
