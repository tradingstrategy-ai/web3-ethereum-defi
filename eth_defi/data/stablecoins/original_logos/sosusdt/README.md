# sosusdt logo sources

soSUSDT is assumed to be a Sonne Finance lending receipt token (soUSDT on Optimism).
The Sonne Finance protocol logo is used as no token-specific logo exists.

## Source

- **Protocol**: Sonne Finance
- **Protocol website**: https://sonne.finance/
- **Logo source**: Sonne Finance GitHub organisation avatar
- **Source URL**: https://avatars.githubusercontent.com/u/111396927?s=400&v=4
- **Format**: PNG, 203×203 px, RGBA with transparency

## Files

- `sosusdt.generic.png` — Sonne Finance brand mark (icon only, orange circle with white sun), used as primary logo
- `sosusdt.wordmark.webp` — Sonne Finance wordmark (icon + "Sonne" text), downloaded from https://sonne.finance/sonne-short.webp

## Notes

- The Sonne Finance site displays a warning that the protocol was exploited and advises users not to interact with it.
- No token-specific logo exists for soSUSDT; the Sonne Finance brand mark is the best available representation.
