# VUSD Hemi logo sources

VUSD is the native stablecoin on the Hemi Network (chain ID 43111), bridged from
Ethereum mainnet (contract `0x677ddbd918637E5F2c79e164D402454dE7dA8619`).

## Files

- `vusd-hemi.generic.svg` — Official SVG logo from the Hemi token list GitHub Pages.
  This is a composite icon showing the VUSD coin badge (purple circle with V$ mark)
  plus a small Hemi network badge in the bottom-right corner. Size: 20×20 viewBox.
- `vusd-hemi.generic.jpg` — 250×250 JPEG from CoinGecko (coin ID: `vusd`).
  Clean purple circle with V+dollar-lines icon, no network badge. Used for post-processing.
