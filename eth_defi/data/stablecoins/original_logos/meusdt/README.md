## Logo decision

No dedicated Rome Finance or meUSDT project logo was found. The Rome Finance project
has no homepage, CoinGecko listing, or Twitter account. No GitHub organisation named
RomeFinance or similar was found.

As a fallback, the USDT (Tether) logo was used, as meUSDT represents a Curve LP
position denominated in USDT.
