# plusd-polyquity (PolyQuity PUSD) — logo sources

## Project

PolyQuity PUSD is a stablecoin issued by PolyQuity, a defunct Liquity fork on Polygon (~$94K TVL).
- Docs: https://docs.polyquity.org/ (may be offline)
- DeFiLlama: https://defillama.com/protocol/polyquity
- GitHub: https://github.com/PolyQuity
- Token contract on Polygon: 0x08B7a8f2B9c675dCe7808B51E82adf240F495fC0 (PUSD)

Note: The project is largely defunct. `polyquity.org` was taken over by a different party
(WordPress site with AI-generated content dated 2026-01). The original project's GitHub org
remains accessible and was used as the logo source.

DeFiLlama icon source: https://icons.llama.fi/polyquity.jpg

## Post-processing

The GitHub org avatar `polyquity-avatar.generic.png` (460×460 RGBA) was used as the source,
as it has an alpha channel and higher resolution than the DeFiLlama JPEG.
The post-processing script detected a dark logo and applied colour inversion
(bright pixels → transparent, dark pixels → white) for display on dark backgrounds.
