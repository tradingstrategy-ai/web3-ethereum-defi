# Franklin Templeton Benji logo sources

## Sources

- **Homepage**: https://digitalassets.franklintempleton.com/benji/
- **Benji full logo**: https://digitalassets.franklintempleton.com/images/benji-logo-full.svg
- **Benji app icon**: https://digitalassets.franklintempleton.com/icons/icon-512x512.png

## Files

- `franklin-templeton.light.svg` — official 163×56 Benji header wordmark, designed for
  the platform's dark footer.
- `franklin-templeton.generic.png` — official 512×512 transparent Benji application
  icon. This square brand mark is used as the post-processing source.

## Processed variants

The source icon is transparent and sufficiently bright for both themes. It was
post-processed into `formatted_logos/franklin-templeton/light.png` and
`formatted_logos/franklin-templeton/dark.png`, each 256×256 RGBA PNG.
