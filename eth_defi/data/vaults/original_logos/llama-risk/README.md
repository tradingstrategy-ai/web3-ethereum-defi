## Notes

- Light variant: fill `#f0ede5` ("Llama Dune") — off-white, suitable for dark backgrounds
- Dark variant: fill `#0c2529` ("Llama Cyan") — dark teal, suitable for light backgrounds
- Both symbols are on transparent backgrounds
- Post-processing used `llama-risk.dark.svg` (dark symbol) as source for `generic.png`
- The post-processing script inverted colours automatically (dark → white) for use on dark backgrounds
