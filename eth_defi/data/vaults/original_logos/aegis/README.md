# Aegis logo

## Source

- Icon-only SVG extracted from the official [Aegis website](https://aegis.im/images/logo.svg) on 2026-07-21.
- The [Aegis homepage](https://aegis.im/) uses this mark in its footer and links to the organisation's official social accounts.

## Files

- `aegis.generic.svg` — 56 × 57 SVG brand mark with a transparent background.
- `aegis.dark.svg` — a dark slate rendering of the same official mark for light backgrounds.

## Notes

The icon-only mark is preferable to the site's horizontal wordmark for the standard 256 × 256 logo output. The official mark is white; `aegis.dark.svg` preserves its geometry while supplying the standard dark-on-light rendering.
