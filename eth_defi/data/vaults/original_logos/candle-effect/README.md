# Candle Effect logo

`candle-effect.generic.png` is the square candlestick brand mark published as
the icon of the official Candle Effect Discord community. The same Discord
invite is published in the descriptions of the Candle Effect v1 and v2
Lighter pools, and Discord identifies the community as “Candle Effect” with a
copy-trading, signals and strategies description.

The guild identity was verified through Discord's public invite API:

https://discord.com/api/v10/invites/PCxhXEj7hR?with_counts=true

The 512×512 PNG was retrieved on 2026-07-20 from:

https://cdn.discordapp.com/icons/924687253459066900/75f19bc7f3d9463a1e1bc5dd7237885d.png?size=512

The stale `@candleffect` X avatar was not used because it currently depicts
an unrelated design shop.
