# Bosera Asset Management (International) logo

The light wordmark was downloaded from the header of the
[official Bosera International website](https://www.bosera.com.hk/en-US/index):

- Source URL: `https://www.bosera.com.hk/en-US/assets/images/common/logo-alpha@2x.png`
- Original format: transparent PNG
- Original dimensions: 256 × 64 pixels
- Saved as: `bosera-asset-management-international.light.png`

No public brand kit or higher-resolution vector brand mark was found. The
official site favicon is square but only 32 × 32 pixels, so the header wordmark
was retained as the better-quality source.

The source was post-processed into the standard 256 × 256 transparent
`formatted_logos/bosera-asset-management-international/light.png` variant for
dark backgrounds.
