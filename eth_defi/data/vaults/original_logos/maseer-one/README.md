# Maseer One logo sources

- `maseer-one.generic.svg` was downloaded from the official Maseer homepage:
  https://maseer.finance/logo-square.svg
- The asset is the square Maseer brand mark displayed in the homepage navigation.
- The official site provided this default variant only; no separate light and dark
  source assets were exposed. `formatted_logos/maseer-one/light.png` is the
  processed variant suitable for light backgrounds.
