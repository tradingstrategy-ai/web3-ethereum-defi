# 3Jane vault protocol logo source

Logo for the 3Jane vault protocol (USD3 / sUSD3 credit vaults).

- `logo.png` — 4000×4000 brand mark (white pixel-art face on 3Jane blue `#0029FF`
  circle), from `https://www.3jane.xyz/logo.png`.
- `logo-mark.svg` — vector brand mark from
  `https://www.3jane.xyz/assets/capital/logo-mark.svg`.

Same asset as the USD3/sUSD3 stablecoin logos under
`eth_defi/data/stablecoins/original_logos/usd3/` (see that folder's README for
full source URLs and brand colour).
