# Frankencoin logo sources

Logo assets for the Frankencoin vault protocol metadata.

## Sources

- Homepage: https://frankencoin.com/
- Token page: https://frankencoin.com/token/
- Brand assets: https://github.com/Frankencoin-ZCHF/www/tree/main/media_kit
- Brand guidelines: https://frankencoin.com/Frankencoin_Brand_Guidelines.pdf

## Files

- `frankencoin.generic.svg` - svZCHF / Frankencoin Savings Vault coin mark from
  the official token page asset
  `https://frankencoin.com/images/Coin_Logo_Frankencoin_saved.svg`.
- `frankencoin.wordmark.webp` - Frankencoin header wordmark from
  `https://frankencoin.com/images/Frankencoin-Logo-Compressify.io.webp`.

## Notes

- The square savings vault coin mark is preferred for formatted vault protocol
  icons because it is a compact brand mark and directly represents svZCHF.
- No separate light and dark icon variants were needed; the same coloured SVG
  mark is usable on both backgrounds.
