# Mellow vault protocol logo sources

Logo sources for the Mellow vault protocol metadata.

- `mellow.generic.svg` - coloured Mellow brand mark from the official media kit:
  https://docs.mellow.finance/resources/media-kit
- `mellow.light.svg` - monochrome brand mark for dark backgrounds from the same media kit.
- `mellow.dark.svg` - monochrome brand mark for light backgrounds from the same media kit.

The official coloured circular brand mark is already square, has transparent
outside corners, and reads on both light and dark backgrounds. The formatted
`light.png` and `dark.png` variants therefore both use `mellow.generic.svg` as
their source.
