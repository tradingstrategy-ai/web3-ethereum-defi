# Nara logo sources

## Source

- Official brand kit: https://www.nara.io/brand
- NaraUSD+ SVG token mark: https://www.nara.io/brand/logos/NaraUSD%2B.svg
- NaraUSD+ PNG token mark: https://www.nara.io/brand/logos/NaraUSD%2B.png

The official brand kit identifies this mark as the NaraUSD+ yield-token logo.

## Files

- `nara.generic.svg` - Official vector NaraUSD+ token mark.
- `nara.generic.png` - Official 293x293 NaraUSD+ token mark.

The PNG is a square icon-only asset and is used to create the formatted vault
protocol logo variants.
