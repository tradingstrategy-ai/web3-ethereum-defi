# KiloEx logo sources

The original logos were downloaded from the official [KiloEx homepage](https://www.kiloex.io/)
on 2026-07-13.

- `kiloex.generic.svg` — `https://www.kiloex.io/assets/static/logo.BDNSFpAu.svg`
- `kiloex.alternative.svg` — `https://www.kiloex.io/assets/static/logo2.DHXqfTaY.svg`
- `kiloex.generic.ico` — `https://www.kiloex.io/favicon.ico` (square brand mark)

These are original website assets. The formatted logo variants are generated
from the square favicon brand mark using `scripts/logos/post-process-logo.py`.
