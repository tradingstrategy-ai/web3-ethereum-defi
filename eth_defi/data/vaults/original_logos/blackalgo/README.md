# Blackalgo logo sources

## Source

- **Brand mark SVG**: Extracted from the official [blackalgo.com](https://blackalgo.com) website favicon
  - Direct URL: `https://blackalgo.com/favicon.svg`
  - The favicon SVG is a 1000×1000 icon with the Blackalgo brand mark (no text), dark background

## Variants

- `blackalgo.generic.svg` — brand mark (icon only), dark background (`#000000`) with light icon elements (`#EDEDF8`, `#C9CAFD`), 1000×1000 viewBox

## Notes

- No separate light/dark variants or brand kit were available on the website
- The homepage also contains a wordmark SVG (120×32 viewBox with icon + text) but the favicon is preferred as it is icon-only
- Post-processed to `formatted_logos/blackalgo/generic.png` (256×256 PNG, transparent background)
