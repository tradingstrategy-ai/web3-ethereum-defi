## Files

- `usds.generic.png` - USDS token logo from Trust Wallet assets (256x256, RGBA, transparent background), sourced for Ethereum contract address 0xdC035D45d973E3EC169d2276DDab16f1e407384F

## Notes

- USDS is the upgraded version of DAI, issued by Sky (formerly MakerDAO), website: https://sky.money/
- The logo is an orange/yellow gradient circle with a white Sky "S" mark
- The same design is used across all major token lists (Uniswap, CoinGecko, Trust Wallet) for the USDS stablecoin
- sky.money CDN has a composite USDS+Ethereum icon (`_USDS_flagship.svg`) but this is a vault product icon, not the token icon
- No dedicated SVG version of the USDS token icon was found; the Trust Wallet PNG at 256x256 RGBA is the best available source
- Twitter/X: https://x.com/SkyEcosystem
