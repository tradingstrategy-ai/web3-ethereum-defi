# Piku logo source

- **Source:** [Piku media assets](https://docs.piku.co/piku/resources/media-assets)
- **Downloaded asset:** `Piku Production.svg`
- **Saved file:** `piku.generic.svg`

The official Piku documentation publishes this square brand mark as part of
its media assets. The standardised `formatted_logos/piku/generic.png` image
was generated from this source using `scripts/logos/post-process-logo.py`.
