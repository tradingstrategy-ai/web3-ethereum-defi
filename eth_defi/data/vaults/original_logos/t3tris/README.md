# T3tris logo sources

## Sources

- Homepage: https://t3tris.finance/
- Official brand assets from homepage static files:
  - https://t3tris.finance/static/logos/brand/t3tris_white_icon.svg
  - https://t3tris.finance/static/logos/brand/t3tris_black_icon.svg
  - https://t3tris.finance/static/logos/brand/t3tris_horizontal_white_logo.svg
  - https://t3tris.finance/static/logos/brand/t3tris_horizontal_black_logo.svg

## Files

- `t3tris.light.svg` - white icon-only brand mark from the homepage favicon asset.
- `t3tris.dark.svg` - black icon-only brand mark from the matching homepage static asset.
- `t3tris.wordmark.light.svg` - white horizontal wordmark from the homepage header.
- `t3tris.wordmark.dark.svg` - black horizontal wordmark from the homepage header.

## Notes

- No separate public brand kit was found.
- The formatted outputs use the icon-only brand marks because the logo workflow
  prefers brand marks over wordmarks for protocol metadata icons.
- `formatted_logos/t3tris/generic.png` uses the white icon so the default
  website logo stays visible on dark backgrounds.
- `formatted_logos/t3tris/dark.png` uses the black icon.
- `formatted_logos/t3tris/light.png` uses the white icon.
