# SingularV logo sources

## Source

Logos were downloaded from the official SingularV brand kit page:
<https://www.singularv.xyz/brand-kit>

## Files

| File | Description |
|------|-------------|
| `singularv.generic.svg` | Coloured symbol (circle icon, #FF4000 orange fill on transparent background) — used for post-processing |
| `singularv.symbol-color.svg` | Original coloured symbol from brand kit (orange #FF4000 fill, with CSS var()) |
| `singularv.symbol-white-circle.svg` | White symbol on transparent background (circle variant) |
| `singularv.symbol-white.svg` | White symbol on transparent background (square variant) |
| `singularv.wordmark-light.svg` | Full wordmark with text, white fill (light variant, 576×61 px) |

## Notes

- The brand kit labels "light" and "dark" variants differently to our convention. Files have been renamed to reflect actual content.
- The `singularv.generic.svg` has the CSS variable `var(--fill-0, #FF4000)` replaced with the hardcoded hex `#FF4000` so that cairosvg can render it correctly.
- The symbol is a circular "V" shape in SingularV's brand orange (#FF4000).
