# Neutrl logo sources

The official [Neutrl brand kit](https://www.neutrl.finance/brand) provides
separate SVG downloads for the logo and its isolated mark. The isolated mark
was selected because it is square and is intended for avatars and favicons.

- `neutrl.dark.svg` — official dark mark for a light background.
- `neutrl.light.svg` — official light mark for a dark background.

Downloaded on 2026-07-21 from the official brand-kit SVG endpoints:

- <https://cdn.neutrl.finance/branding/logo/svg/neutrl-mark-dark.svg>
- <https://cdn.neutrl.finance/branding/logo/svg/neutrl-mark-light.svg>

Neutrl's [domain migration announcement](https://www.neutrl.finance/blog/domain-migration-neutrl-finance)
identifies `neutrl.finance` as its sole official web domain.
