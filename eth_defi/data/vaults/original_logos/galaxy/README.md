# Galaxy logo sources

## Source

- **Website**: https://www.galaxy.com
- **Twitter**: https://x.com/GalaxyHQ
- **Logo extracted from**: https://www.galaxy.com (homepage, inline SVG in site header)

## Files

| File | Description |
|------|-------------|
| `galaxy.generic.svg` | Brand mark (icon-only) extracted as inline SVG from the Galaxy homepage header. Uses `currentColor` fill — renders black by default. |
| `galaxy.light.svg` | Explicit white brandmark variant for dark backgrounds. |
| `galaxy.dark.svg` | Explicit black brandmark variant for light backgrounds. |

## Notes

- No brand kit or press/media page was found at `/brand`, `/brand-kit`, `/media`, `/press`, or `/assets`.
- The logo is the Galaxy "G" brand mark: a square with rounded top corners forming a distinctive bracket/square shape with two path elements.
- The explicit variants avoid relying on `currentColor` during SVG rasterisation,
  which previously produced an unusable formatted image.
