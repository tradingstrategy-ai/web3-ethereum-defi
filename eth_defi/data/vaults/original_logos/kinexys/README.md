## Files

- `kinexys.dark.png` - Official Kinexys dark-background logo from the homepage Open Graph image.
- `kinexys.light.png` - Official Kinexys light-background logo from the site footer.
- `kinexys.favicon.png` - Official square Kinexys brandmark from the site icon metadata.

## Notes

- No separate brand kit, press kit, GitHub repository, or SVG logo was found.
- The wide source logos include the Kinexys wordmark. The formatted metadata logo uses the official favicon because it is the only square icon-only source exposed by the Kinexys site.
- The favicon has a transparent edge and dark built-in background. It was post-processed into the standard 256x256 PNG `light.png` and `dark.png` variants.
