# August Digital logo sources

## Sources

- **Homepage**: https://www.augustdigital.io
- **Icon mark (SVG)**: https://www.augustdigital.io/_next/static/media/august-logo-black.0bd609fc.svg (52x52, dark rounded-square background with white "A" diamond motif)
- **Wordmark (SVG)**: https://www.augustdigital.io/_next/static/media/august-logo-text-black.39451b96.svg (94x29, black text wordmark "August")

## Files

- `august-digital.generic.svg` - Icon mark (copy of icon.svg). 52x52, dark (#111214) rounded square with white diamond/arrow logo. Used as the primary square logo.
- `august-digital.icon.svg` - Icon mark SVG. 52x52, dark background with white "A"-shaped diamond motif.
- `august-digital.wordmark.svg` - Text wordmark SVG (94x29). Full "August" text in dark (#2E323F) colour.

## Notes

- No dedicated brand kit or press page was found on the website.
- The icon mark is preferred as the generic logo because it is already a square format and works as a brandmark.
- Post-processing inverted the icon colours (dark pixels became transparent, making the white diamond visible on dark UI backgrounds).
