# Fidelity logo sources

- Official Fidelity International careers website: https://careers.fidelityinternational.com/
- Official full-colour SVG wordmark: https://careers.fidelityinternational.com/static_cms/fil_site_assets/images/fil-logo.svg
- Official square brand mark: https://careers.fidelityinternational.com/static_cms/fil_site_assets/images/f-logo-mobile.png

The source SVG is a transparent, full-colour Fidelity International wordmark with a 153 × 48 view box. It is preserved without conversion or other post-processing as `fidelity.generic.svg` for the shared Fidelity curator identity.

The source PNG is the 179 × 179 full-colour brand mark used by the official mobile website. It is preserved as `fidelity.generic.png` and selected for standardisation because its square format remains legible in compact logo placements.
