# Theo logo sources

- `theo.generic.svg` is Theo's square brand mark extracted from the official
  [Theo homepage](https://theo.xyz/) on 2026-07-17. The asset is served by the
  homepage's official Framer CDN at
  `https://framerusercontent.com/images/VJ0hHk1je6Lx7L1QvmuaGLL3UuQ.svg`.
- `theo.light.svg` is Theo's white word mark from that same official homepage.
  It is retained as an original source, but the square brand mark is used for
  the formatted protocol and curator logo.

The official brand-kit link in Theo's documentation returned 404 during this
extraction, so the live homepage assets are the documented source.
