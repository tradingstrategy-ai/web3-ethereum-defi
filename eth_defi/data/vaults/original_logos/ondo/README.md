# Ondo logo sources

- `ondo.generic.svg` is the official square Ondo mark served by the Ondo
  homepage at <https://ondo.finance/icon.svg>.

Downloaded on 2026-07-17. The homepage provides a single black, transparent
SVG mark rather than separate theme variants. The post-processed `light.png`
is the inverted version for dark backgrounds; no independent official dark
variant was available.
