## Alternative versions available

The brand kit contains additional variants that were not downloaded:
- Logos on blue background
- Additional PNG exports at various sizes

## File naming convention

- **light**: Light/white text or elements on dark or transparent background
- **dark**: Dark/black text or elements on light or transparent background
- **generic**: Generic version that works in multiple contexts
- **wordmark**: Logo with brand name text
- **brandmark**: Logo mark only without text
- **full**: Complete logo with both mark and text
