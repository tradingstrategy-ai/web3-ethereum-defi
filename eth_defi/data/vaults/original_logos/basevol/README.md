# BaseVol logo extraction

## Sources

- **Favicon SVG** (contains embedded PNG): https://basevol.com/favicon.svg
- **Twitter avatar**: https://x.com/BaseVolApp (via unavatar.io)

## Files

- `basevol.favicon.svg` - Original SVG favicon (wraps base64 PNG, 702KB)
- `basevol.generic.png` - Extracted PNG from favicon (1024x1024, RGBA, transparent background)
- `basevol.twitter.png` - Twitter avatar (200x200, RGBA, black background)

- The BaseVol logo is a blue baseball cap (3D rendered style)
- Best source is the favicon-extracted PNG at 1024x1024 with transparent background
