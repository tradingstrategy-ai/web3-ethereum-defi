# Midas logo sources

Midas is a platform for composable onchain investment products.

- Website: https://midas.app/
- Documentation: https://docs.midas.app/
- Official links page: https://docs.midas.app/resources/official-links
- X: https://x.com/midasrwa

## Logo files

- `midas.generic.svg` - first-party wordmark exposed by the Midas access-restricted homepage.
  - Source: https://stg.midas.app/full.svg
- `midas.twitter.jpg` - square avatar from the official Midas X account, used as the source for the formatted icon.
  - Source: https://unavatar.io/x/MidasRWA

## Notes

- `midas.generic.svg` is a wide wordmark and is kept as a first-party source reference.
- `midas.twitter.jpg` is the official X avatar for the account linked from the Midas documentation's official links page. It is more suitable for a square 256x256 formatted logo than the wide wordmark.
