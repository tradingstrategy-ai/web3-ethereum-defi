# Sentinel Codes logo

`sentinel.wordmark.svg` is the official Sentinel combination mark served by
the [Sentinel Codes website](https://sentinel.codes/). It is a 134×32
transparent SVG.

`sentinel.generic.svg` preserves the official vector artwork but crops its
view box to the standalone 32×32 brand mark for the curator avatar. No paths
or colours were changed.

Source URL: https://sentinel.codes/logo.svg
