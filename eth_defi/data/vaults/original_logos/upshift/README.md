# Upshift logo sources

## Sources

- Homepage: https://www.upshift.finance/
- Documentation: https://docs.upshift.finance/
- 512x512 favicon / app icon: https://cdn.prod.website-files.com/69fced2dc843fe30f0a4d650/6a333a805c6b8e661adc8e23_favicon.png
- Documentation wordmark: https://docs.upshift.finance/

## Files

- `upshift.brandmark.png` - 512x512 RGBA favicon/app icon from the official
  Upshift website. This is the preferred source for formatted vault logos
  because it is square and contains only the brand mark.
- `upshift.generic.svg` - White and green Upshift wordmark from the official
  documentation. Kept as a source reference, but not used for formatted logos
  because the vault logo convention prefers brandmarks without text.

## Notes

- The previous formatted logos used the wide wordmark, which became cropped in
  a 256x256 icon frame.
- `formatted_logos/upshift/generic.png`, `dark.png`, and `light.png` are all
  generated from `upshift.brandmark.png`.
