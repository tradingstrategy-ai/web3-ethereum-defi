# AlphaGrowth logo sources

AlphaGrowth is a DeFi growth and curator platform that curates Euler EVK vaults.

- Website: https://alphagrowth.io/
- Twitter: https://x.com/alphagrowth1
- GitHub: https://github.com/alphagrowth

## Logo files

- `android-chrome-384x384.png` (384x384, best for icon use) - stylised green "alpha" symbol on black background
  - Source: https://alphagrowth.io/static/images/android-chrome-384x384.png?1
- `twitter-avatar.jpg` (400x400) - same green "alpha" symbol, from Twitter profile
  - Source: https://pbs.twimg.com/profile_images/1970196035730309121/JMUg_PVs_400x400.jpg
- `header-logo.svg` - white wordmark "alphagrowth" text logo (147x20, used in site header)
  - Source: https://alphagrowth.io/static/branding/v2/header-logo.svg
- `alphagrowth-logo-inline.svg` - larger grey wordmark "alphagrowth" text logo (1400x195)
  - Source: https://alphagrowth.io/static/branding/v2/alphagrowth-logo-inline.svg

## Notes

- The primary brand mark is a stylised green gradient "alpha" (Greek letter) symbol on a dark background
- The `android-chrome-384x384.png` is the best file for a square icon/avatar
- The SVG wordmarks contain only the text "alphagrowth" without the icon mark
- GitHub organisation avatar was a generic blue pixel pattern (not the AlphaGrowth brand), so it was excluded
