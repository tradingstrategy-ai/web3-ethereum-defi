# Cassa logo

`cassa.generic.svg` is the white Cassa wordmark used in the navigation of
the [official Cassa website](https://www.cassa.fyi/). The site does not
publish a separate square brand mark, favicon, media kit, or alternative logo
asset, so the official wordmark is retained and padded by the standard logo
processor.
