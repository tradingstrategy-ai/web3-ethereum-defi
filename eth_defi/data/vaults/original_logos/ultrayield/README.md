# UltraYield logo sources

## Source

Logo extracted from the UltraYield website apple-touch-icon and favicon assets.

- Website: https://ultrayield.app
- Twitter: https://x.com/ultrayieldapp

## Files

- `ultrayield.generic.png` — 180x180 RGBA PNG (apple-touch-icon from https://ultrayield.app/apple-touch-icon.png)
- `ultrayield.generic.apple-touch-icon.png` — Original apple-touch-icon download (same as above)
- `ultrayield.generic.svg` — SVG favicon from https://ultrayield.app/favicon.svg (contains base64-encoded raster PNG, not a clean vector)

## Notes

No brand kit or press page is available at the time of extraction (2026-05-05).
The site is a JavaScript-rendered app; logos were obtained via HTML meta tags.
The SVG favicon contains a white background and embedded raster PNG — the PNG files are preferred for post-processing.
