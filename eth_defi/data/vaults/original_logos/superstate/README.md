# Superstate logo sources

- `superstate.generic.svg` is the official square brand mark served as the
  [Superstate favicon](https://superstate.com/favicon.svg), downloaded on
  2026-07-17.
- `superstate.dark.png` is the official dark wordmark image embedded on the
  [Superstate homepage](https://superstate.com/) at
  `https://assets.superstate.com/images/2025/09/29/superstate-black-logo.png`,
  downloaded on 2026-07-17.

The square favicon is the preferred brand-mark source for the light variant.
The wordmark is retained as the official dark-source reference; the standard
processor inverts it for the dark-background output.
