# Byzantine Finance logo

`bizantine.wordmark.svg` is the official Byzantine Finance combination mark
used in the header of the [Byzantine Finance website](https://www.byzantine.fi/).
It is a 180×57 transparent SVG containing the purple brand mark and wordmark.

`bizantine.generic.svg` preserves the official vector artwork but crops its
view box to the standalone 57×57 brand mark for the curator avatar. No paths
or colours were changed.
