# SwissBorg logo sources

## Source

- **Brand mark SVG**: Extracted from the inline SVG embedded in the SwissBorg homepage navigation
  - Direct URL: `https://swissborg.com`
  - Element: `<svg id="logo" viewBox="0 0 1193.5 1168.7">` in page HTML
  - CSS classes `st1` and `st4` sourced from page stylesheet and inlined into the SVG

## Variants

One variant was extracted:

- `swissborg.generic.svg` — brand mark (icon only), geometric hexagonal circuit-board icon in SwissBorg green (`#35b98a`) on transparent background, viewBox `0 0 1193.5 1168.7`

## Notes

- No dedicated brand/press download page was available (brand kit is behind a Google Drive link)
- The inline SVG from the homepage contains the complete brand mark geometry
- CSS style classes were inlined to make the SVG self-contained
- Post-processed to `formatted_logos/swissborg/generic.png` (256×256 PNG, transparent background)
