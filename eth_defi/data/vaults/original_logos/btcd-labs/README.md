# BTCD Labs logo source

- Official brand image: <https://btcd.fi/_next/static/media/btcd-logo-text.55ee2523.png>
- Downloaded: 2026-07-17

The official image is a wide Bitcoin Dollar wordmark. `btcd-labs.generic.png`
contains its left-hand square BTCD brandmark, extracted without alteration so
it remains legible in the curator-icon slot. The standard post-processing
pipeline produced `formatted_logos/btcd-labs/light.png` for dark charts.
