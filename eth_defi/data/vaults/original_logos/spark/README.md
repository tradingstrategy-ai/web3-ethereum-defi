# Spark logo sources

## Source

- **Brand kit**: https://docs.spark.fi/brand
- **Download**: https://drive.google.com/drive/folders/1mknRFyo4VfVpAT-Lhh6-hGbxRnJ1RnAY

## Files

| File | Description | Dimensions | Source |
|------|-------------|------------|--------|
| `spark.generic.svg` | Original logo (light background) | 106x32 | docs.spark.fi |
| `spark.light.svg` | Full logo for light backgrounds (dark text) | 2003x550 | Brand kit ZIP |
| `spark.dark.svg` | Full logo for dark backgrounds (white text) | 2003x550 | Brand kit ZIP |
| `spark.logomark.svg` | Icon only (gradient star) | 497x497 | Brand kit ZIP |

## Notes

- The dark background version (`spark.dark.svg`) has white text with the pink/orange gradient star
- The light background version (`spark.light.svg`) has dark text with the same gradient star
- The logomark is just the gradient star icon without text
- All logos use the same pink-to-orange gradient for the star icon
