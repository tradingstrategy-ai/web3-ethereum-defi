# Aave logo sources

- `aave.generic.png` — official Aave ghost brand mark (200×200, transparent RGBA).
  Sourced from the canonical TrustWallet asset for the AAVE token
  (`0x7Fc66500c84A76Ad7e9c93437bFc5Ac33E2DDaE9`):
  https://raw.githubusercontent.com/trustwallet/assets/master/blockchains/ethereum/assets/0x7Fc66500c84A76Ad7e9c93437bFc5Ac33E2DDaE9/logo.png

The ghost is a mid-tone gradient brand mark that reads on both light and dark
backgrounds, so the same source is used for the `generic`, `light` and `dark`
post-processed variants.

Note: the Aave homepage (https://aave.com/) brand kit could not be scraped
directly because the Claude-in-Chrome browser tooling was unavailable during
extraction; the TrustWallet asset is the official token logo and an acceptable
substitute. Replace with the v4 brand kit mark if a higher-resolution source
becomes available.
