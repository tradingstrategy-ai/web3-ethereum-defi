# Epoch RWA logo

The original assets were extracted from the official [Epoch RWA website](https://www.epochrwa.com/):

- `epoch-rwa.generic.ico` is the square, transparent favicon used by the site.
- `epoch-rwa.generic.svg` is the vector logo artwork published at [`/Logo.svg`](https://www.epochrwa.com/Logo.svg). The source artwork is a cropped, low-opacity decorative variant, so the square favicon is used for the formatted logo.

The 256x256 formatted PNGs were generated from the favicon with `scripts/logos/post-process-logo.py`.
