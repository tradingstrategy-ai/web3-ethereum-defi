# CNCB Capital logo

The original square CNCB Capital mark was extracted from the header of the
official CNCB Capital fund website.

- Source page: https://www.cncbinvestment.com/fund.asp?pageid=79
- Original asset: https://www.cncbinvestment.com/img/logo2.png
- Format: transparent PNG, 70 × 70 pixels
- Saved as: `cncb-capital.generic.png`

The official site also provides a horizontal wordmark, but the square mark is
used here because it is better suited to the repository's curator logo format.
