# Yuzu Money logo sources

The Yuzu Money brand mark was extracted from the official Yuzu Money website.

| File | Source | Notes |
|------|--------|-------|
| `yuzu-money.generic.png` | `https://yuzu.money/assets/yuzu.png` | Official lime-and-gold citrus brand mark on a dark circular field; 72 x 72 PNG with transparency. |

The website header also serves an official word mark at
`https://yuzu.money/assets/logo.png`. The icon-only asset above was selected
because it is square and is more suitable for the standardised logo output.
No public brand kit or GitHub logo asset was found on the official website.

The formatted `generic.png`, `light.png`, and `dark.png` assets are 256 x 256
PNG versions generated from this official brand mark.
