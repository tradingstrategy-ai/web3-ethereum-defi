# Sygnum logo source

`sygnum.generic.png` is the avatar served by Sygnum's official GitHub
organisation, [`sygnumbank`](https://github.com/sygnumbank). It is used because
the official Sygnum website returned a Cloudflare challenge to the automated
logo collector on 2026-07-17. The formatted light and dark PNG files were
created from this original with `scripts/logos/post-process-logo.py`.
