# Bulla Network logo sources

The original logo was downloaded from the official
[Bulla Network homepage](https://www.bulla.network/) on 2026-07-21.

- `bulla.generic.svg` — `https://www.bulla.network/favicon/favicon.svg`
  (111×112 px SVG favicon containing Bulla's orange brand mark).

The official site exposes this single coloured brandmark rather than separate
light and dark variants. It was processed with
`scripts/logos/post-process-logo.py` into the transparent 256×256 `generic`,
`light` and `dark` PNG assets. The same orange mark remains legible on both
chart backgrounds.
