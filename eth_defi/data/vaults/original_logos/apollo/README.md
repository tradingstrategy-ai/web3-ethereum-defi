`apollo.generic.png` is the square Apollo “A” brandmark cropped from the
left-hand mark of the official wordmark. It is post-processed into the local
256×256 dark-chart asset at `formatted_logos/apollo/light.png`.

This slug is Apollo Global Management, used by the current vault JSON for
Ember Apollo ACRED. It is deliberately distinct from the existing
`apollo-crypto` curator.
