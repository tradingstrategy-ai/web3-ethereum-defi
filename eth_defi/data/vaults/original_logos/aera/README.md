# Aera vault protocol logo source

Logo assets for the Aera vault protocol.

## Sources

- Homepage: https://www.aera.finance/
- Brand kit: https://www.aera.finance/brand-kit
- Full brand guide: https://www.figma.com/proto/lBN5xqyRP6EnhI60aEKQbJ/Aera-Design-Library-%26-Guidelines

## Saved originals

- `aera.wordmark.light.svg` - full Aera wordmark for dark backgrounds, sourced from the homepage header logo.
- `aera.wordmark.dark.svg` - full Aera wordmark for light backgrounds, sourced from the brand kit header logo.
- `aera.logomark.light.svg` - lime logomark for dark backgrounds, downloaded from the Aera brand kit.
- `aera.logomark.dark.svg` - blue logomark for light backgrounds, downloaded from the Aera brand kit.
- `aera.generic.png` - 256x256 favicon/logomark from the Aera website.

The formatted outputs use the square logomark variants because the logo
post-processing workflow prefers brand marks over wordmarks for protocol icons.
