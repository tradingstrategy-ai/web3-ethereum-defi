# Utorg logo sources

Official source: Utorg website.

- Homepage: https://utorg.com/
- `utorg.generic.svg`: compact Utorg header logo SVG embedded in the official homepage navigation.

The SVG was extracted from the official homepage inline logo markup.

The white transparent post-processed mark was kept as
`formatted_logos/utorg/light.png`. `generic.png` and `dark.png` use the same
mark recoloured dark so it remains visible on light backgrounds.
