## Logo description

The Fluid logo is a stylised circular icon featuring:
- A wave/water motif through the centre
- Two-tone blue colour scheme (light blue #4B8EF5 / dark blue #1E3A8A approximately)
- The design resembles a split circle with a flowing wave connecting the halves
- Transparent background
