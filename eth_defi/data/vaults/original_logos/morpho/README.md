## Terminology

- **dark mode**: Light/white graphics designed for use on dark backgrounds
- **light mode**: Dark/black graphics designed for use on light backgrounds
- **symbol**: The butterfly icon without text
- **horizontal/vertical**: Wordmark layouts with "Morpho" text

## Brand information

The Morpho brand is inspired by the Blue Morpho butterfly, symbolising evolution and metamorphosis. The brand uses a signature blue and black colour palette with grain-style 3D visuals.
