# TAU (628 Labs) logo sources

TAU is a vault curator associated with 628 Labs.

- Homepage: https://www.628labs.xyz
- Twitter: https://x.com/628Labs

## Variants

- `tau.light.svg` — wordmark + icon for light backgrounds (dark `#111216` text and orange `#FB740A` icon), downloaded from https://www.628labs.xyz/logo-text.svg
- `tau.dark.svg` — wordmark + icon for dark backgrounds (white `#F6F6F6` text and orange `#F18F00` icon), downloaded from https://www.628labs.xyz/logo-text-white.svg

## Notes

- Both SVGs include the TAU "TAURUS" wordmark and the geometric 3D-diamond icon mark
- The icon uses a two-tone design: a solid geometric shape in dark/white plus an orange accent
- No standalone icon-only SVG was available on the website
- Post-processed to `formatted_logos/tau/light.png` and `formatted_logos/tau/dark.png` (256×256 PNG, transparent background)
