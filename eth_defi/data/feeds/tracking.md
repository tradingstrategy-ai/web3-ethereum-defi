## Conventions

| Symbol | Meaning |
|--------|---------|
| ⬜ | To do — no YAML file created yet |
| ✅ | Done — YAML file written |
| ⚠️ | Skipped — no active feeds found |

- `eth_defi/data/feeds/protocols/{slug}.yaml`
