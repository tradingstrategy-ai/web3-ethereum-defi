Royco API
---------

`Royco <https://royco.org/>`__ vault protocol integration.

Royco is an incentivised action market protocol. This integration supports
Royco ERC-4626 wrappers and senior/junior tranche vaults that expose custom
``AssetClaims`` accounting.
