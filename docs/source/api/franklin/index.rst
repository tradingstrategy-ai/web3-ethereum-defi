Franklin Templeton Benji API
============================

The :mod:`eth_defi.tokenised_fund.franklin` package contains the read-only
adapter for reviewed Ethereum BENJI and iBENJI fund-share contracts. It uses
the issuer's `Benji contract registry <https://digitalassets.franklintempleton.com/benji/benji-contracts/>`__
as the authoritative address source.
