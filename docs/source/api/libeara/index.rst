Libeara API
============

The adapter covers the Ethereum CMTAT CUMIU and BELIF shares and the Arbitrum
ULTRA deployment. CMTAT products expose issuer-maintained NAV, while ULTRA is
reported as supply-only because it has no verified public NAV/share source.
