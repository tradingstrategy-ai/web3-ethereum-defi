Uniswap v2 DEX API
------------------

Uniswap v2 API contains Python helpers to interact with Uniswap v2 style exchanges like:

- Uniswap v2 on Ethereum mainnet

- Sushi

- QuickSwap

- Trader Joe

- PancakeSwap
