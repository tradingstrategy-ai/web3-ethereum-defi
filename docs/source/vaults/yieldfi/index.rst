YieldFi API
-----------

`YieldFi <https://yield.fi/>`__ protocol integration.

YieldFi is an asset management platform for the onchain economy. The protocol actively
deploys capital into risk-managed, high-performing DeFi strategies across stablecoins,
ETH, and BTC asset classes, delivering competitive APY to depositors.

The platform uses vyToken vaults (e.g., vyUSD) that implement ERC-4626 with additional
features including configurable vesting periods for reward distribution and guardrail
mechanisms for risk management.
