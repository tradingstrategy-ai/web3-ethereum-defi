Liquid Royalty API
------------------

Liquid Royalty is a vault product on `Berachain <https://www.berachain.com>`__
that uses USDe as the underlying asset. It is part of the `Liquidity Royalty <https://www.liquidroyalty.com>`__
protocol family, which implements tiered vault architecture
with profit spillover and cascading backstop mechanisms.

The ALAR SailOut Royalty vault is an ERC-4626 compliant vault
deployed on Berachain.
