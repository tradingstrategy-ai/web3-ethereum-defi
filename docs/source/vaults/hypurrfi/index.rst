HypurrFi API
------------

`HypurrFi <https://www.hypurr.fi/>`__ integration.

HypurrFi is a lending market built on HyperEVM (Hyperliquid) that enables users to deposit and borrow
native Hyperliquid assets for leveraged yield strategies.

The protocol offers pooled and isolated lending markets where lenders can earn yield by providing liquidity
to borrowers seeking leveraged exposure to Hyperliquid-native assets.
