ZeroLend API
------------

`ZeroLend <https://zerolend.xyz/>`__ integration.

ZeroLend is a multi-chain DeFi lending protocol built on Layer 2 solutions, based on Aave V3.
It specialises in Liquid Restaking Tokens (LRTs) lending, Real World Assets (RWAs) lending,
and account abstraction.

ZeroLend uses `Royco Protocol's <https://royco.org/>`__ WrappedVault infrastructure for
incentivised ERC-4626 vault wrappers with integrated rewards systems. This allows ZeroLend
vaults to offer additional reward programmes alongside the underlying vault yield.
